# 🐟 고래미 원가계산 프로그램

고래미 수산가공식품제조회사를 위한 체계적인 원가계산 프로그램입니다.

## 📋 주요 기능

### 1. 📦 원료비 계산
- 배합비율 기반 원료 투입량 계산
- 원료별 단가 및 총 비용 계산
- 평균 단가 및 총 투입량 산출

### 2. 📦 포장비 계산
- 포장재료별 단가 및 수량 입력
- 포장재료별 총 비용 및 무게 계산
- 포장비 상세 내역 제공

### 3. 👥 노무비 계산
- 직접인건비 (생산직)
- 간접인건비 (관리직)
- 일용직 인건비
- 노무비 구성 비율 시각화

### 4. 🏭 제조경비 계산
- 기타제조경비
- 복리후생비 (노무비 대비 비율)
- 감가상각비 (자산가치, 사용연한 기반)
- 제조경비 구성 분석

### 5. 💰 원가계산
- 총 제조원가 계산
- 개당 원가 산출
- 이윤률 기반 가격 설정
- 도매가격 계산

### 6. 📈 경영실적 분석
- 매출액 및 매출원가 분석
- 매출총이익 및 순이익 계산
- 수익성 지표 (매출총이익률, 순이익률)
- 경영실적 시각화

## 🚀 설치 및 실행

### 1. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행
```bash
streamlit run cost_calculator.py
```

## 📊 사용법

### 기본 설정 (사이드바)
- **제품명**: 계산할 제품의 이름
- **생산수량**: 생산할 제품의 개수
- **기준 투입량**: 원료 계산의 기준이 되는 투입량 (kg)
- **이윤률**: 제조원가 대비 이익률 (%)
- **경영비용**: 판관비, 영업외비용, 법인세율 설정

### 원료비 계산
1. 기본 원료 6개가 미리 설정되어 있습니다
2. 배합비율, 단가를 입력하면 자동으로 계산됩니다
3. 추가 원료가 필요한 경우 개수를 설정하여 추가할 수 있습니다

### 포장비 계산
1. 기본 포장재료 2개가 설정되어 있습니다
2. 단가, 수량, 개당 무게를 입력하면 총 비용이 계산됩니다
3. 추가 포장재료도 설정 가능합니다

### 노무비 계산
1. **직접인건비**: 생산직 근로자의 인건비
2. **간접인건비**: 관리직 근로자의 인건비
3. **일용직**: 임시 근로자의 인건비
4. 각각 일일생산량 비율, 시급, 근무시간, 근로자 수를 입력합니다

### 제조경비 계산
1. **기타제조경비**: 기준 제조경비와 일일생산량 비율로 계산
2. **복리후생비**: 노무비의 일정 비율로 계산
3. **감가상각비**: 자산가치, 사용연한, 생산일수로 계산

### 원가계산
- 모든 비용이 자동으로 합산되어 총 제조원가가 계산됩니다
- 개당 원가와 총 원가를 모두 확인할 수 있습니다
- 이윤률에 따른 예상판매가와 도매가격이 계산됩니다

### 경영실적 분석
- 매출액, 매출원가, 매출총이익, 순이익을 분석합니다
- 수익성 지표와 차트를 통해 경영상황을 파악할 수 있습니다
- 결과를 JSON 파일로 저장할 수 있습니다

## 🔧 커스터마이징

### 새로운 원료 추가
`cost_calculator.py` 파일의 `default_materials` 리스트에 새로운 원료를 추가할 수 있습니다:

```python
default_materials = [
    {"name": "새로운원료", "ratio": 1.0, "unit_price": 1000},
    # ... 기존 원료들
]
```

### 새로운 포장재료 추가
`default_packaging` 리스트에 새로운 포장재료를 추가할 수 있습니다:

```python
default_packaging = [
    {"name": "새로운포장재료", "unit_price": 100, "quantity": 1000, "weight_per_unit": 0.1},
    # ... 기존 포장재료들
]
```
